# 🕹️ Unity Game Assignment – רכיבים, תנועה ומצלמות

פרויקט זה נוצר כחלק מהקורס לפיתוח משחקי מחשב.  
המטרה הייתה להכיר את סביבת Unity דרך יצירת סצנות, רכיבים בסיסיים, תנועה, וסימולציית מצלמה.

---

## 🎯 מטרות המטלה
1. להכיר את מערכת הסצנות של Unity.
2. ליצור רכיבים שמפעילים **תנועה** ו־**רוטציה** על עצמים.
3. להבין כיצד מצלמות עובדות במשחק – זוויות מבט, מעקב אחרי שחקן, ושינוי שדה ראייה.
4. לבנות משחקון קצר שמדגים את השימוש ברכיבים והמצלמות.

---

## 🧱 רכיבים שפיתחנו

| שם הרכיב | תיאור | מה הוא עושה |
|---------|-------|-------------|
| **Oscillator** | תנועה מחזורית | גורם לעצם לזוז בין שתי נקודות וחוזר חלילה (תנועה סינוסואידלית) |
| **Rotator** | סיבוב של עצם סביב ציר | מסובב את העצם במהירות קבועה סביב ציר X/Y/Z |
| **Pulse** | מדמה לב פועם | מגדיל מקטין את העצם בהדרגה |
| **Visability** | העלמת עצם | מעלים את העצם ומחזיר אותו על ידי לחיצת כפתור(H) |

כל אחד מהרכיבים נבנה עם **פרמטרים מתכווננים** ב־Inspector, כך שניתן לשלוט במהירות, מרחק ותדר.

---

## 🎥 מצלמות ונקודות מבט

בסצנה הוספנו **מצלמה עוקבת** שמלווה את השחקן מלמעלה (Top-Down).  
התאמנו:
- `orthographicSize` כדי לשלוט בגודל שדה הראייה.
- מיקום המצלמה כך שתשמור זווית יציבה.
- מפת־מיניאטורה (Mini-map) שמציגה מבט על הסביבה.

קישור למדריך שהשתמשנו בו:  
https://docs.unity3d.com/Manual/class-Camera.html

## 🎮 איך להריץ את המשחק
1. פתחו Unity (גרסה 2021+ מומלצת).
2. Clone או הורידו את המאגר.
3. פתחו את התיקייה בפרויקט Unity.
4. הריצו את הסצנה הראשית (`MainScene`).
5. להעברה לסצנה 2 (חלק ב של המטלה) לחץ 2.
6. חזרה לסצנה 1 (חלק א של המטלה) לחץ 1.
